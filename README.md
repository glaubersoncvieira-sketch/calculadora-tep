# Calculadora TEP — Biocor Rede D'Or

Ferramenta de suporte à decisão clínica para Tromboembolismo Pulmonar (TEP).

## Funcionalidades

- **Calculadora Manual**: Escores Wells, PERC, sPESI, Hestia + Classificação AHA/ACC 2026
- **Análise por IA**: Cola história clínica em texto livre → IA extrai dados e classifica automaticamente

## Acesso

🌐 **https://[usuario].github.io/calculadora-tep/**

## Configuração da Análise por IA

Na página de Análise por IA, clique em "⚙️ Configurar Chave API" e insira sua chave da OpenAI.
A chave fica salva apenas no seu navegador (localStorage) — não é enviada para nenhum servidor.
